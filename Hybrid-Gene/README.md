# Hybrid Gene

RPG Maker MZ game files.

---

## Structure

```
Hybrid-Gene/
├── audio/          # Music and sound effects
├── data/           # JSON database files
├── fonts/          # Custom fonts
├── img/            # Graphics (tilesets, sprites, UI)
├── js/             # Plugins and core scripts
└── movies/         # Video cutscenes
```
