Overview

Data structure standards for monsters, moves, items, and game systems in Hybrid Gene. These structures map to RPG Maker MZ's database with custom extensions for breeding, growth, and AI systems.

Data Structures

Monster 
{
  id: Number,
  name: String,
  species: "insect" | "mammal" | "reptilian" | "avian",
  generation: 1 | 2 | 3 | 4,

  birthmark: {
    personality: String,
    coreAI: "aggressive" | "defensive" | "supportive" | "evasive",
    statModifiers: { str, int, agi, end }  // -0.2 to +0.2
  },

  stats: { str, int, agi, end },
  derivedStats: { hp, maxHp, energy, maxEnergy, energyRegen, moveSpeed },
  moves: [ 4 basic, 4 special ],
  temperament: Number,  // 0-100
  obedience: Number,    // hidden, calculated
  parents: [MonsterId, MonsterId] | null
}
Move 
{
  id: Number,
  name: String,
  category: "physical" | "magic" | "buff" | "debuff" | "heal" | "aoe",
  energyCost: Number,
  cooldown: Number,     // seconds
  damage: Number,       // base multiplier
  effects: [EffectId],
  range: Number,
  aoe: Number | null
}
Birthmark 
{
  id: Number,
  name: String,
  personality: String,
  coreAI: "aggressive" | "defensive" | "supportive" | "evasive",
  statModifiers: {
    str: Number,  // -0.2 to +0.2
    int: Number,
    agi: Number,
    end: Number
  }
}

Formulas

Obedience Calculation
obedience = baseObedience + (INT * 0.4) + (Temperament * 0.6)
Stat Inheritance (Breeding)
childStat = (parent1.stat + parent2.stat) / 2 * (0.8 + random(0.4))
Stat Growth (Action-based)
gain = baseGain * birthmarkModifier * decayMultiplier
Growth Decay
decayMultiplier = 1 / (1 + stat / decayThreshold)
