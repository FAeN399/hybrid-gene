File Naming Convention

[species]_[generation]_[name]_[state].png

Examples

Filename Species Gen State
insect_g1_beetle_idle.png Insect G1 Idle
mammal_g2_wolfcat_attack.png Mammal G2 Attack
reptilian_g3_drake_ko.png Reptilian G3 KO

Valid Values

Species: insect mammal reptilian avian

Generations: g1 g2 g3 g4

States: idle attack hit ko special
