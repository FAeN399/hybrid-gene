Hybrid Gene

Game Design Document

Version 1.0 December 2024 Pre-Production

1. Executive Summary

Vision Statement

Hybrid Gene is a real-time monster breeding and battle arena game where players raise, breed, and command semi-autonomous creatures. Unlike traditional monster-catching games with turn-based combat, Hybrid Gene features fluid real-time battles where monsters have their own intelligence and may choose to ignore commands.

Core Fantasy

Breed. Train. Battle.

The player is a commander and coach - not a direct controller. Monsters are partners with opinions.

Target Experience

  • 40-60 hours to Grand Champion
  • Strategic but fast-paced
  • Rewards planning and reflexes

Key Differentiators

Real-Time Combat

No menus, no turns. Position, timing, and reaction matter.

Monster Autonomy

Low INT = can't understand orders. Low Temperament = won't obey. Commands can be ignored.

Generational Breeding

G1 real animals to G3 mythological to G4 secret deity-tier hybrids.

Action-Based Growth

No XP or levels. Stats grow through battle actions with diminishing returns.

Reference Games

  • Digimon World 1 - Monster autonomy, breeding depth
  • Link to the Past - Real-time combat feel, fluid movement

2. Core Loop

EXPLORE > CAPTURE > BREED > TRAIN > BATTLE > ARENA

Arena rewards unlock new regions. Repeat with stronger monsters.

Breeding Rules

  • Same species + Same generation = Next generation offspring
  • G3 cross-species breeding = Secret G4 creatures
  • Example: Avian G3 + Reptile G3 = Quetzalcoatl

3.3 Birthmarks

Every monster is born with a random birthmark that permanently shapes development:

  • Personality - Flavor text and minor behaviors
  • Core AI Archetype - Aggressive / Defensive / Supportive / Evasive
  • Stat Modifiers - -20% to +20% growth rate per stat

Birthmarks cannot be changed. Breeding for ideal birthmarks is part of the strategic depth.

4.2 Growth Decay

No Levels - Stats grow directly from battle actions.

Diminishing Returns - As stats increase, growth rate decreases.

5. Breeding System

Eligibility

  • Same species required
  • Same generation required
  • Long cooldown after breeding

Inheritance

  • Stats: Formula from parents with variance
  • Moves: Inherited if child can learn
  • Birthmarks: Fresh random generation
ChildStat = ((Parent1 + Parent2) / 2) x (0.8 + Random(0.4))

6. Combat System

6.1 Battle Format

Format

  • 100% Real-time combat
  • 1v1, 1v2, and 2v2 modes
  • Player controls 2 monsters
  • Top-down / 2.5D camera

Monster Autonomy

  • Hidden obedience = f(INT, Temperament)
  • Low INT = can't understand orders
  • Low Temperament = won't obey
  • Ignored commands = Core AI takes over

6.2 Controls

D-Pad
Stance orders: Aggressive / Defensive / Supportive / Passive
L1 / R1
Select Monster 1 / Monster 2, access move commands
A B X Y
Execute moves (while holding L1 or R1)
L1+L2 / R1+R2
Advanced options, switch to special moves
L2 / R2
Use items on monsters

6.3 Move System

  • 8 slots: 4 basic + 4 special
  • Categories: Physical, Magic, Buff, Debuff, Heal, AoE
  • Energy: Regenerates but slows over battle duration
  • Learning: Parents, stat thresholds, watching others

6.4 Command Feedback

Green flash = Command accepted

Red flash = Command ignored (Core AI activates)

Hit stagger + screen shake + sound effects for impact feel.

7. Capture & Taming

Tranquilize

  • Monster must be below 25% HP
  • 50% success at 25% HP
  • 100% success at 5% HP or less
  • Can wake up if battle continues

Carry Back

  • Movement speed halved
  • Can't use items in battle
  • Death = lose the monster
  • Wild monsters start with low temperament
